# wallpaper_app


A desktop app that allows you preview and favorite in-app wallpapers

To install on your windows machine: 

1. Extract the folder from the zip file, 
2. Click on wallpaper.exe in the extracted folder,
3. Continue to follow the installation steps as spelt out by your machine.
